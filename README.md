# Number-Guessing-Game-
This is an amazing console project.
This is a Number Guessing game, which uses Random and Scanner.

Programming Language : JAVA

Game Flow : A. Secret number is fixed by system ( Random).
            B. User will guess the number.
            C. On the correct guess, attempts and score will be shown.

Concepts Used   : Loops, OOPS(Constructor), Methods.
